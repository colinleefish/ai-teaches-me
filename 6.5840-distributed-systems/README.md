# 6.5840 Distributed Systems (Spring 2025)

MIT's graduate-level distributed systems course covering the fundamentals of building reliable, scalable distributed systems.

## Course Overview

**Schedule:** TR 1:00-2:30pm  
**Source:** [MIT PDOS](https://pdos.csail.mit.edu/6.824/schedule.html)

## Topics Covered

- **Core Concepts:** MapReduce, RPC, threads, consistency models, fault tolerance
- **Replication & Consensus:** Primary-backup replication, Raft consensus algorithm
- **Distributed Storage:** GFS, ZooKeeper, Spanner
- **Transactions:** Two-phase commit, optimistic concurrency control (FaRM)
- **Advanced Topics:** Cache consistency (Memcached), serverless (AWS Lambda), Byzantine fault tolerance, blockchain (Bitcoin)

## Structure

- **Labs:** Hands-on implementation of MapReduce, key-value servers, Raft, and sharded systems
- **Exams:** Mid-term (March) and final (May)
- **Optional:** Final project with team presentations

## Learning Goals

Build practical understanding of distributed systems through implementing real-world algorithms and studying production systems from Google, Amazon, Facebook, and others.
