# ai-teaches-me

A personal learning repository where AI teaches me various DevOps/SRE topics through structured lessons and hands-on practice.

## Topics Covered

### Go Programming (Summer 2025)
Comprehensive Go learning path from basics to advanced topics:
- **Phase 1-2**: Basics, variables, types, control flow, functions
- **Phase 3**: Structs and interfaces
- **Phase 4**: Concurrency patterns (goroutines, channels, sync primitives)
- **Phase 5**: Standard library (file I/O, JSON, HTTP, context)
- **Phase 6**: Web servers and databases
- **Phase 7**: Testing
- **Phase 8**: Optimization and deployment
- **Project**: GMDB (Go Movie Database API)

### Apache Kafka (Fall 2025)
Docker-based Kafka learning environment with comprehensive notes covering:
- Configuration and setup
- Topics and partitions
- Producers and consumers

### Grafana Alloy (Fall 2025)
Telemetry pipeline configuration and observability:
- Alloy configuration language
- Prometheus integration
- Docker deployment setup
