# ai-teaches-me

A personal learning repository where AI teaches me various DevOps/SRE topics through structured lessons and hands-on practice.

## Topics Covered

### Go Programming (Summer 2025)
Comprehensive Go learning path from basics to advanced topics:
- **Phase 1-2**: Basics, variables, types, control flow, functions
- **Phase 3**: Structs and interfaces
- **Phase 4**: Concurrency patterns (goroutines, channels, sync primitives)
- **Phase 5**: Standard library (file I/O, JSON, HTTP, context)
- **Phase 6**: Web servers and databases
- **Phase 7**: Testing
- **Phase 8**: Optimization and deployment
- **Project**: GMDB (Go Movie Database API)

### Apache Kafka (Fall 2025)
Docker-based Kafka learning environment with comprehensive notes covering:
- Configuration and setup
- Topics and partitions
- Producers and consumers

### Grafana Alloy (Fall 2025)
Telemetry pipeline configuration and observability:
- Alloy configuration language
- Prometheus integration
- Docker deployment setup

### Kubernetes (Spring 2026)
Operational Kubernetes for SREs — scheduling, availability, networking, and security:
- Scheduling, affinity, taints, topology spread
- PodDisruptionBudgets, rolling updates, probes
- Networking, ingress, network policies
- RBAC, Workload Identity, secrets management
- Resource management, HPA/VPA, cost optimization
- Observability and debugging patterns

### Helm (Spring 2026)
Kubernetes package manager — chart authoring, templating, and release lifecycle:
- Charts, templates, and Go template functions
- Values layering and schema validation
- Dependencies, subcharts, and library charts
- Hooks, tests, and release lifecycle
- Packaging, OCI registries, and CI/CD patterns

### 6.5840 Distributed Systems (Winter 2025)
MIT's graduate-level course on distributed systems:
- MapReduce, RPC, fault tolerance, consensus (Raft)
- Distributed storage (GFS, Spanner) and transactions
- Production systems from Google, Amazon, Facebook
- Labs implementing real distributed algorithms
